# 問題1 小金持ち

たかし君は1円硬貨をA枚、5円硬貨をB枚、10円硬貨をC枚、50円硬貨をD枚、100円硬貨をE枚、500円硬貨をF枚持っています。X円の商品を買うとき、計算の得意なたかし君はなるべくお釣りが少なくなるように支払います。たかし君が支払う硬貨の種類と枚数を求めてください。  

### 制約
A≦4  
B≦1  
C≦4  
D≦1  
E≦4  
F≦1  

### 入力
入力は以下の形式です。
```
A B C D E F  
X
```

### 出力
```
A B C D E F
```

---
### 入力例
```
3 0 2 1 0 1
423
```

### 出力例
```
3 0 2 0 0 1
```
