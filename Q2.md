# 問題2 X

長さNの数列Sと、整数Xがある。数列の要素(A,B)が、X=A×Bとなるような組み合わせの数を調べるアルゴリズムを考えてください。また、数列は昇順に並んでいて、数列の要素は自然数です。

### 制約
<pre>
1 < N < 10<sup>5</sup>  
0 < X < 10<sup>4</sup>  
0 < A < 10<sup>5</sup>  
0 < B < 10<sup>5</sup>
</pre>

### 例
```
N = 7
X = 10
数列
{ 1, 2, 3, 5, 6, 8, 10, }
```

### 答え
```
2
```
(1, 10)と(2, 5)の二通りがあります。
